# TEKAL

Landing page comercial para TEKAL, un software punto de venta para restaurantes, cafeterias y fast food.

## Enfoque

- Sistema POS con licencia permanente.
- Sin mensualidades obligatorias.
- Instalacion y soporte segun el plan contratado.
- Conversion principal por WhatsApp.

## Personalizacion

- Cambia el telefono de WhatsApp en `index.html`.
- WhatsApp configurado: `https://wa.me/524641163543`.
- Actualiza el correo `contacto@tekal.com` si usas otra cuenta.
- Ajusta precios o condiciones comerciales en la seccion `precios`.

## Publicacion

El sitio es estatico. Puede publicarse en Cloudflare Pages, GitHub Pages o cualquier hosting que sirva archivos HTML, CSS y JS.
