Preguntas

Preguntas frecuentes

El sistema cobra mensualidad?

No. La licencia es permanente. Algunos servicios adicionales como soporte extendido, nuevas implementaciones o actualizaciones futuras pueden depender del plan contratado.

Necesito internet para usarlo?

No necesariamente. El sistema puede trabajar en red local dentro del negocio. Algunas funciones como soporte remoto, descarga de actualizaciones o validacion de licencia pueden requerir internet.

Incluye instalacion?

Si. Segun el plan contratado, se incluye instalacion guiada o configuracion remota para dejar el sistema listo para operar.

Funciona con impresora de tickets?

Si. Se puede configurar con impresoras compatibles para tickets, comandas de cocina o barra.

Sirve para restaurante y tienda?

Si. TEKAL cuenta con soluciones para restaurante, cafeteria, fast food, tienda, boutique, abarrotes y negocios similares.

Puedo pedir cambios o funciones especiales?

Si. Las modificaciones a medida se pueden revisar segun el alcance tecnico, tiempo requerido y plan contratado.
