FAQ

Preguntas frecuentes antes de implementar

TEKAL cobra mensualidad?+

No. El enfoque comercial principal es licencia permanente con un solo pago.

Funciona con comandera movil?+

Si. El ecosistema contempla comandera Android para enviar pedidos directo a cocina o barra.

Sirve para mesas, mostrador y pedidos para llevar?+

Si. El flujo esta pensado para operar salon, caja, barra y pick-up desde una misma base.

Incluye instalacion o ayuda inicial?+

Segun el plan, incluye instalacion guiada, configuracion remota y acompanamiento de arranque.

Necesito internet para operar diario?+

No de forma constante. La operacion local puede mantenerse y el acceso remoto se usa cuando aplica.

Arquitectura recomendada

Separacion clara entre sitio comercial y soporte para clientes

La home debe vender el ecosistema. Las descargas, versiones y activaciones deben vivir fuera del flujo comercial para no generar ruido ni exponer detalles tecnicos de entrada.
